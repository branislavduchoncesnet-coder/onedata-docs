---
title: "How do I request space in Onedata?"
icon: Single
---

To be able to use Onedate, you need to have access to Space. This can be either shared with you or granted access to.  
If new space is needed, request can be created via email onedata@e-infra.cz. Tempalte with all the necessary info can be found bellow:  

*Hello Team,*  
  
*I request Onedata Space with capacity of `<space_capacity>`GB.*  
*Project lead:*  
*Project members:*  
*Organization name:*  
*Project name:*  
*Project description:*
